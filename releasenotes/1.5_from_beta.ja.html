CotEditor 1.5 の 1.5β1 からの変更点

リリース: 2014-05-08

新機能

  • 「グリフ情報を表示」機能
    • 1文字だけ選択した状態で Cmd + Option + I もしくは「編集」メニューから「グリフ情報を表示」を実行すると、選択している文字の情報がポップオーバー
  • 「文字を Unicode 16 進数で入力」機能を「編集」メニューに追加
  • “CoffeeScript”, “Dart”, “Diff”, “INI”, “Makefile”, “reStructuredText”, “Textile”, “YAML” シンタックス定義を追加

機能改善／仕様変更

  • { と } の間で改行したときはインデントを深めて } 前でも改行・インデントするように変更
  • “Perl” シンタックス定義の更新
    • 数値, 変数のカラーリング
    • いくつかのキーワードの追加
    • コメントカラーリングの改善
  • ステータスバーとドロワーの行数情報に選択された範囲の行数を追加
  • ビューの不透明度を5%の倍数以外でも設定できるように変更
  • 情報ドロワーの Unicode 表示でサロゲートペアに対応
  • シンタックスカラーリングにおいて抽出条件が正規表現でないとき、ローマ数字とアンダースコア _ は文字列が継続しているとみなすように変更
  • 対応する括弧のハイライトをよりわかりやすいものに変更
  • スクリプトエラーパネルに関する変更
    • エラーメッセージ内の検索機能を実装
    • テキストを少し大きくし、エラー毎に空行を入れた
  • 入力補完リスト設定の選択肢に「シンタックススタイルからのみ」を追加
  • OgreKit framework を 2.1.6 に更新
    • 検索パネルが常に現在のデスクトップスペースに表示されるように
  • ほか細部とドキュメントの変更

バグフィックス

  • [1.5β1] シンタックス定義名を編集したときに、変更が大文字／小文字の入れ替えのみだとアラートが出て保存できない不具合を修正
  • [1.5β1] タブ展開の設定を変更したときに、変更がフォーカスのあるビューにしか適応されない不具合およびツールバーアイコンの状態が更新されないことがある不具合を修正
  • [1.5β1] 日本語環境で、スクリプトエラーパネルが表示されない不具合を修正
  • [1.5β1] 環境設定のショートカットが消えていた不具合を修正
  • [1.5β1] 入力補完リストの設定が「一般英単語から」を含むとき、補完リストが出ない不具合を修正
  • [1.5β1] 文字列の置換の Undo/Redo でカラーリングが更新されない不具合を修正
  • [1.5β1] シンタックススタイル編集シートの「出荷時に戻す」ボタンの有効／無効の制御をより厳密に
  • [1.5β1] ファイルドロップの設定ができない不具合を修正
  • [1.5β1] プリントパネルで設定変更をしたときに本文表示領域がずれることがある不具合を修正

  • [1.5β1] 日本語へのローカライズ漏れを修正
  • 書類が空の状態で入力補完リストを出そうとするとコンソールにエラーが出る不具合を修正
